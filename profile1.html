Background

​

Jessica Chan is living proof that you don't require a computing science degree or need to splurge on an intensive coding boot camp in order to learn web development (although if you have/want to do any of these things then go for it!). She is a strong advocate that you can become a web developer at any age, career stage and transition from any work background with enough patience, persistence and willingness to teach yourself how to code. Find out how Jessica became a web developer below: ​


How I learned to code and landed a job (no CS degree!)

​ ​

Alternatively, you can read her story here (link: https://www.freecodecamp.org/news/how-i-learned-to-code-without-a-cs- degree-or-bootcamp/)​

Why Is She Known? Impacts/Works

​ ​

Jessica has gone on to not only work as a professional web developer after teaching herself how to code but is now a full-time Youtube content creator with the goal to help beginner web developers learn how to code. She provides a wealth of free resources under the name Coder Coder, both on her website and Youtube channel. Watch her video below to see how you can get started: ​
